------Readme---------

- This project involves generating different formats of output like xml,json
  from the data obtained throught the queries on a database;
  
- The queries are written to achieve the following directives in the output:
    --> A list of all suppliers and their parts
    --> A list of parts ordered by cost (highest to lowest) 

- "database.sql" contains the query for creating a table with the structure for
   which the application was designed.
